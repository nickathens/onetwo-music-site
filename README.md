# onetwo-music.com

The OneTwo Music website. A single-page, client-side build (`index.html` + `data.js` + `assets/`), served via GitHub Pages on the custom domain `onetwo-music.com`.

OneTwo Music is a bespoke film-music production house based in Athens.

## Structure

- `index.html` — the entire site (markup, styles, and the interaction/audio engine, inlined)
- `data.js` — work items, films, team, and contact data
- `assets/` — brand mark, score audio, the film reel, posters, and thumbnails

No build step. Editing `index.html` or `data.js` and pushing to `main` redeploys.
